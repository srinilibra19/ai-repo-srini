## Key Design Decisions
| Decision | Why |
|----------|-----|
| tcps://localhost:55443 for Solace | mTLS required per backlog AC; certs generated by generate-certs.sh |
| authentication-scheme: CLIENT_CERTIFICATE | Starter does not auto-switch to mTLS from keystore presence alone |
| client-name includes destination + ${HOSTNAME:dev} | Prevents session takeover with concurrent local sessions |
| ?sslmode=prefer on JDBC URL | Avoids plaintext default; local container accepts prefer; RDS upgrades |
| ${ENV_VAR:default} for all credentials | No naked credential literals per CLAUDE.md security standards |
| IRSA design documented in comments | AwsConfig.java must use StaticCredentialsProvider only when property non-blank |
